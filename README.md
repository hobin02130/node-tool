## ✨ 主要功能 (Features)

* **可视化**: 可视化的数据仪表盘
* **轻量化**: 无复杂的后端程序，256mb内存即可跑起来
* **部署简单**: 可直接二进制文件启动
* **多端支持**: 完美win、linux及多架构。

---

## 🛠️ 技术栈 (Tech Stack)

* **后端**: Python (Flask)
* **前端**: HTML5
* **数据库**: SQLite / PostgreSQL
